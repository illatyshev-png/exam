# Memory - exam.ilat.info

## 📋 Проект

Landing page для курса подготовки к экзамену в Минюсте (групповая подготовка)

**Технологии**: React 18 + TypeScript + Vite + Tailwind CSS + shadcn/ui
**Домен**: https://exam.ilat.info
**Деплой**: cPanel FTP через GitHub Actions

## 📁 Ключевые файлы

- `src/content/landing.ts` - весь контент landing page
- `src/App.tsx` - маршрутизация (6 страниц)
- `vite.config.ts` - Vite конфигурация
- `public/.htaccess` - редиректы (HTTPS, /exam → exam.ilat.info)

## 🛠 Скрипты

```bash
npm run dev              # Dev сервер: localhost:8080
npm run build            # Продакшн + react-snap (prerendering)
npm run build:no-snap    # Без prerendering (для деплоя)
npm run test             # Запуск тестов
```
