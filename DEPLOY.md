## Что загружать

Загружайте **все содержимое папки `dist`** на ваш хостинг.

## Инструкция по загрузке

### Вариант 1: FTP/SFTP (для большинства хостингов)

1. Подключитесь к хостингу через FTP-клиент (FileZilla, WinSCP и т.д.)
2. Найдите корневую директорию сайта (обычно `public_html`, `www`, `htdocs` или `web`)
3. Загрузите **ВСЕ** файлы и папки из папки `dist` в корневую директорию
4. Убедитесь, что файл `index.html` находится в корне

### Вариант 2: Панель управления хостингом

1. Войдите в панель управления хостингом
2. Найдите файловый менеджер
3. Перейдите в корневую директорию сайта
4. Загрузите все файлы из папки `dist`

### Вариант 3: SSH (для продвинутых пользователей)

```bash
# На локальной машине
cd dist
scp -r * user@your-server.com:/path/to/public_html/
```

## Важные моменты

### 1. Настройки сервера

Файл `.htaccess` уже включен для Apache серверов и содержит:
- Перенаправление на HTTPS
- Настройки кэширования
- Правила для SPA (Single Page Application)

### 2. Для Nginx

Если ваш хостинг использует Nginx, добавьте в конфигурацию:

```nginx
location / {
    try_files $uri $uri/ /index.html;
}
```
