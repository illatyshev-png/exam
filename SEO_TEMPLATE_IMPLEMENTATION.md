## ⚠️ Что осталось сделать (критично)

### 1. Prerendering / SSR 🔴 ВЫСОКИЙ ПРИОРИТЕТ

**Проблема:** React SPA может плохо индексироваться поисковиками.

**Решения:**

#### Вариант A: react-snap (быстро)
```bash
npm install --save-dev react-snap
```

Обновить package.json:
```json
{
  "scripts": {
    "build": "vite build && react-snap"
  },
  "reactSnap": {
    "source": "dist",
    "minifyHtml": {
      "collapseWhitespace": true,
      "removeComments": true
    },
    "puppeteerArgs": ["--no-sandbox"]
  }
}
```

#### Вариант B: Next.js (долго, но лучше)
Миграция на Next.js с SSG/SSR для идеальной индексации.

#### Вариант C: Prerender.io (платный сервис)
Динамический рендеринг для ботов.

**Рекомендация:** Начать с react-snap (вариант A).
