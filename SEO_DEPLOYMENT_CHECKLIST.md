## 📤 Сразу после деплоя

### 1. Проверка доступности файлов (5 минут)

- [ ] Открыть `https://exam.ilat.info/sitemap.xml` - должен открыться XML
- [ ] Открыть `https://exam.ilat.info/robots.txt` - должен быть текст
- [ ] Открыть `https://exam.ilat.info/manifest.json` - должен быть JSON
- [ ] Открыть главную страницу и проверить title в браузере

### 2. Проверка мета-тегов (10 минут)

- [ ] Открыть исходный код страницы (Ctrl+U)
- [ ] Найти `<title>` - должно быть "Подготовка к экзамену в Минюсте | 60% сдают с 1 раза"
- [ ] Найти `<meta name="description"` - должно быть описание
- [ ] Найти `<meta property="og:` - должно быть минимум 7 тегов
- [ ] Найти `<script type="application/ld+json">` - должно быть 5 блоков

### 3. Проверка структурированных данных (10 минут)

- [ ] Открыть [Google Rich Results Test](https://search.google.com/test/rich-results)
- [ ] Вставить URL: `https://exam.ilat.info`
- [ ] Нажать "Проверить URL"
- [ ] Убедиться, что найдены: FAQPage, Course, Organization
- [ ] Проверить, что нет критических ошибок

### 4. Проверка Open Graph (5 минут)

- [ ] Открыть [Facebook Debugger](https://developers.facebook.com/tools/debug/)

- [ ] Нажать "Отладка"
- [ ] Убедиться, что отображается:
  - Заголовок
  - Описание

- [ ] Нажать "Очистить кэш и повторить"

### 5. Проверка изображений (5 минут)

- [ ] Открыть главную страницу

## 🐛 Если что-то пошло не так

### Sitemap не индексируется

**Симптомы:** В Search Console статус "Не удалось получить"

**Решение:**
1. Проверить доступность: `curl https://exam.ilat.info/sitemap.xml`
2. Проверить формат XML (должен быть валидным)
3. Проверить robots.txt (должна быть строка с Sitemap)
4. Подождать 24-48 часов

### Structured Data не отображается

**Симптомы:** Google Rich Results Test не находит данные

**Решение:**
1. Открыть исходный код страницы
2. Найти `<script type="application/ld+json">`
3. Скопировать JSON и проверить на [JSONLint](https://jsonlint.com/)
4. Если JSON валиден - подождать 2-4 недели

### Open Graph не работает

**Симптомы:** При публикации в соцсетях нет превью

**Решение:**
1. Проверить мета-теги в исходном коде
2. Использовать Facebook Debugger для очистки кэша
3. Проверить доступность изображения
4. Убедиться, что изображение > 200x200px

### Страница не индексируется

**Симптомы:** `site:exam.ilat.info` ничего не находит

**Решение:**
1. Проверить robots.txt - должен разрешать индексацию
2. Проверить meta robots - не должно быть noindex
3. Запросить индексирование через Search Console
4. Подождать 3-7 дней
