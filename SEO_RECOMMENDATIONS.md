## 📋 Дополнительные рекомендации

### 1. Server-Side Rendering (SSR) - КРИТИЧНО
**Проблема:** Сайт на React (SPA) может плохо индексироваться поисковиками.

**Решения:**
- **Вариант A (рекомендуется):** Мигрировать на Next.js с SSR/SSG
- **Вариант B:** Использовать prerendering (например, react-snap, prerender.io)
- **Вариант C:** Настроить dynamic rendering для ботов

**Приоритет:** 🔴 ВЫСОКИЙ

### 2. Оптимизация изображений
**Текущее состояние:** Изображения в JPG/PNG формате

**Рекомендации:**
```bash
# Конвертировать изображения в WebP
npm install sharp
# Создать скрипт для автоматической конвертации
```

**Что сделать:**
- Конвертировать все изображения в WebP с fallback на JPG
- Добавить responsive images с srcset
- Оптимизировать размеры (сейчас могут быть слишком большие)
- Использовать CDN для изображений

**Приоритет:** 🟡 СРЕДНИЙ

### 3. Производительность (Core Web Vitals)
**Проверить:**
- LCP (Largest Contentful Paint) - должен быть < 2.5s
- FID (First Input Delay) - должен быть < 100ms
- CLS (Cumulative Layout Shift) - должен быть < 0.1

**Инструменты для проверки:**
- Google PageSpeed Insights
- Lighthouse
- WebPageTest

**Приоритет:** 🟡 СРЕДНИЙ

### 4. Контент и семантика
