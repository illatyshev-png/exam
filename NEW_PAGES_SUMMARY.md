## 📋 Чеклист для react-snap

При установке react-snap нужно добавить все новые страницы:

```json
{
  "reactSnap": {
    "include": [
      "/",
      "/about",
      "/pricing",
      "/faq",
      "/reviews",
      "/author"
    ]
  }
}
```

**Важно:** Без этого новые страницы не будут prerendered!
